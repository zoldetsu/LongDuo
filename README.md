# LongDuo

LongDuo - индивидуальный проект, который повзоляет смотреть сериалы. Осущетсвлена регистрация пользователя с целью добавления избранных сериалов, а также их коментирование.

## Технологии:

- React
- JavaScript
- TypeScript
- Zustand
- React Query
